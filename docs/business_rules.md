# Lista de Tarefas

O sistema **TODO-LIST-FLET** é uma lista de tarefas cross-platform para
agendamento de tarefas baseado em usuários.

Cada usuário pode criar uma conta e gerir suas tarefas.

Onde para criar a conta basta ter:

* nome
* e-mail
* senha

Para acessar o sistema, basta fazer um login usando:

* e-mail
* senha

As tarefas são ações e/ou lembretes que o usuário precisa.

Cada tarefa é composta por:

* Título
* Descrição (Opcional)
* Data a ser executada
* Data que foi criada

As tarefas são vinculadas a seus usuários.

Os usuários podem:

* Criar tarefas
* Ver Tarefas
* Editar Tarefas
* Deletar Tarefas
* Filtrar por Data
* Filtrar pela Semana Atual
* Filtrar pelo Mês Atual
